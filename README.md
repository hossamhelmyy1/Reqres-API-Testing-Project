# Reqres API Testing Project

## Description
This project demonstrates API testing using the Reqres public API. It includes testing endpoints for user creation, retrieval, update, and deletion to validate API functionality and responses.

## What I Worked On
- Imported and organized the Reqres Postman collection.
- Conducted manual and automated API tests.
- Validated response codes, body data, and error handling.
- Documented test scenarios and results.

## Tools & Technologies
- Postman
- Reqres API
- JSON for test data

## Notes
This repository is for showcasing API testing skills. The project is not deployed and does not require live execution.
